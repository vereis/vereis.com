# vereis.com

This is the source code for [my personal site](https://vereis.com), a full-stack
~~wiki/blog~~ _digital garden_ powered by Elixir + Phoenix + Absinthe (API) and
Astro (web) for SSR and SSG.

- [Nix](https://nixos.org/download.html) with flakes enabled

### Setup

